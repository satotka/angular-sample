# angular-sample

- fshop is tutorial with angular(v.8): https://angular.jp/start
- mytest is sample for using angular(v.7)

## usage

change current directory.

```bash
cd mytest
```

install package via npm.

```bash
npm install
```

check angular version.

```bash
npx ng version
```

start(ng serve).

```bash
npm start
```
